# Admission-page-for-student-using-html


## 📖 Description

This project is a simple **Student Admission Form webpage** created using **HTML**.
It allows students to enter their personal details such as name, age, gender, email, course selection, and other information required for admission.

The form is displayed in the **center of the page using an HTML table layout**.

---

## 🛠 Technologies Used

* **HTML5** – for creating the structure of the form
* **HTML Form Elements** – input fields, radio buttons, select menu, textarea, and file upload

---

## ✨ Features

* Collects **student personal details**
* Includes **radio buttons for gender selection**
* Allows **date of birth selection**
* Provides **course selection using dropdown menu**
* Supports **photo upload**
* Simple **submit button for form submission**
* Organized layout using **HTML tables**

---

## 📋 Form Fields

The admission form includes the following fields:

* Full Name
* Age
* Gender (Male / Female / Other)
* Date of Birth
* Email
* Contact Number
* Address
* Course Selection
* Previous Qualification
* Percentage / Marks
* Upload Photo

---

## ▶ How to Run

1. Copy the HTML code.
2. Save the file as **admission_form.html**.
3. Open the file in any web browser (Chrome, Edge, Firefox, etc.).
4. The **Student Admission Form** will appear on the screen.

---

## 📷 Output

The webpage displays a **structured admission form** where students can fill in their details and submit the form.
